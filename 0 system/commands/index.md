# Commands

- `new project <slug>` -> `./0 system/scripts/new_project.sh <slug>`
- `new task <title>` -> create from `task.md` template in `1 todo`
- `promote task <file> <review|archive>` -> `./0 system/scripts/promote_task.sh <file> <state>`
- `weekly review` -> `./0 system/scripts/weekly_review.sh`
- `sync insights` -> run insight extraction and hub sync flow
